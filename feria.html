MI GRANJA

Sistema Inteligente de Control y Monitoreo IoT

🎯 ¿Qué es Mi Granja?

Sistema automatizado que revoluciona la gestión agrícola mediante tecnología IoT, permitiendo el control remoto y monitoreo en tiempo real de variables críticas en tu granja.

💡 Innovaciones Destacadas

🤖
Control Inteligente con IA

Algoritmos adaptativos que aprenden y optimizan automáticamente el consumo energético y recursos.

🔄
Sistema de Histéresis

Control inteligente de bomba que evita ciclos innecesarios, prolongando vida útil del equipo.

⚡
PWM Dinámico

Regulación progresiva de temperatura mediante control PWM, reduciendo consumo hasta 70%.

📱
Alertas Instantáneas

Notificaciones Telegram en tiempo real para eventos críticos desde cualquier lugar del mundo.

🚀 Características Principales

🌡️ Monitoreo Ambiental

📊
Sensor DHT22

Medición precisa de temperatura y humedad con precisión profesional.

🎯
Control Automático

Ajuste automático de calefacción según rangos de temperatura óptimos (20°C - 28°C).

💧 Gestión de Recursos

📏
Sensores Ultrasónicos

Medición continua y precisa de niveles de agua y alimento sin contacto.

⚙️
Automatización Total

Bomba de agua con activación automática al detectar niveles críticos (<30%).

☁️ Conectividad Cloud

🔥
Firebase Realtime

Sincronización instantánea de datos con base de datos en la nube.

🌐
Acceso Remoto

Consulta y control desde cualquier dispositivo con conexión a internet.

🛡️ Seguridad y Confiabilidad

💾
Configuración Persistente

Credenciales almacenadas en EEPROM con checksum de validación.

🔐
Portal Captivo

Configuración WiFi inicial segura mediante interfaz web intuitiva.

⚙️ Tecnología Implementada

  • ESP32 (WiFi + Bluetooth)
  • Sensor DHT22 de precisión
  • Sensores Ultrasónicos HC-SR04
  • Control PWM para eficiencia
  • Firebase Realtime Database
  • Telegram Bot API
  • WebServer embebido

🎓 Aplicaciones

🐔 Granjas Avícolas: Control ambiental para pollos y aves de corral.

🐷 Porcicultura: Gestión de temperatura y agua en cocheras.

🌱 Invernaderos: Monitoreo de condiciones para cultivos protegidos.

🐰 Pequeñas Especies: Control para conejos, cuyes y otros animales.
